# 学生生活习惯调查问卷功能开发计划

## 需求概述
为学生宿舍管理系统添加生活习惯调查问卷功能，学生只能填写一次问卷，管理员可查看所有学生的问卷结果。简化设计：只使用一张表存储学生习惯档案，问卷题目定义为Java常量。字段使用字典类型。

---

## 一、数据库设计

### 1.1 学生习惯档案表 (my_student_habit)
| 字段名 | 类型 | 说明 |
|--------|------|------|
| habit_id | BIGINT | 记录ID (主键) |
| student_id | BIGINT | 学生ID (关联my_student表，唯一) |
| sleep_habit | VARCHAR(50) | 作息习惯字典 (dormitory_sleep_habit) |
| sleep_time | VARCHAR(50) | 睡觉时间字典 (dormitory_sleep_time) |
| wake_time | VARCHAR(50) | 起床时间字典 (dormitory_wake_time) |
| nap_habit | VARCHAR(50) | 午休习惯字典 (dormitory_nap_habit) |
| bath_frequency | VARCHAR(50) | 洗澡频率字典 (dormitory_bath_frequency) |
| clean_importance | VARCHAR(50) | 卫生重视程度字典 (dormitory_clean_importance) |
| tolerate_dirty | VARCHAR(50) | 能否接受室友不打扫字典 (dormitory_tolerate_dirty) |
| study_location | VARCHAR(50) | 学习地点字典 (dormitory_study_location) |
| friend_visit | VARCHAR(50) | 欢迎室友带朋友字典 (dormitory_friend_visit) |
| temperature_preference | VARCHAR(50) | 空调温度偏好字典 (dormitory_temperature) |

## 三、问卷题目定义（Java常量）

题目定义为 `HabitQuestionConstants.java` 常量类：

| 题目编码 | 题目内容 | 类型 | 选项 | 分类 | 是否必填 |
|----------|----------|------|------|------|----------|
| sleep_habit | 您的作息习惯属于？ | single | 早起型,正常型,晚起型 | 作息 | 是 |
| sleep_time | 您一般几点睡觉？ | single | 21:00前,21:00-22:00,22:00-23:00,23:00后 | 作息 | 是 |
| wake_time | 您一般几点起床？ | single | 06:00前,06:00-07:00,07:00-08:00,08:00后 | 作息 | 是 |
| nap_habit | 午休习惯？ | single | 每天午休,偶尔午休,从不等休 | 作息 | 否 |
| bath_frequency | 您的洗澡频率是？ | single | 每天,每两天,每周,看情况 | 清洁 | 是 |
| clean_importance | 您对宿舍卫生的重视程度？ | single | 非常重要,重要,一般 | 清洁 | 是 |
| tolerate_dirty | 您能接受室友不打扫卫生吗？ | single | 完全接受,勉强接受,不能接受 | 清洁 | 否 |
| study_location | 您喜欢在宿舍学习还是去图书馆？ | single | 宿舍,图书馆,都可以 | 社交 | 否 |
| friend_visit | 您欢迎室友带朋友来宿舍吗？ | single | 欢迎,无所谓,不喜欢 | 社交 | 否 |
| temperature_preference | 您对空调温度的偏好？ | single | 怕热(喜欢低温),正常,怕冷(喜欢高温) | 其他 | 否 |
| snoring | 您有打呼噜吗？ | single | 是,否 | 其他 | 否 |

## 四、后端开发

### 4.1 实体类 (Domain)
- `StudentHabit.java` - 学生习惯档案实体

### 4.2 常量类
- `HabitQuestionConstants.java` - 问卷题目常量定义

### 4.3 Mapper层
- `StudentHabitMapper.java` + `StudentHabitMapper.xml`

### 4.4 Service层
- `IStudentHabitService.java` + `StudentHabitServiceImpl.java`

### 4.5 Controller层
- `StudentHabitController.java` - 学生问卷API
  - `GET /susheguanli/studentHabit/list` - 获取学生问卷列表(管理员)
  - `GET /susheguanli/studentHabit/{studentId}` - 获取某个学生的问卷答案
  - `POST /susheguanli/studentHabit` - 提交问卷答案(学生)
  - `PUT /susheguanli/studentHabit` - 更新问卷答案(学生)
  - `GET /susheguanli/studentHabit/my` - 获取当前学生的问卷答案
