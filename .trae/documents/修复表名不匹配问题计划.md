# 修复表名不匹配问题

## 问题分析
代码中使用的是 `sys_dormitory_building`，但数据库中表名已改为 `my_dormitory_building`，导致查询失败。

## 修复步骤

### 1. 查找需要修改的文件
搜索所有引用 `sys_dormitory_building` 的 Mapper XML 文件和 Java 文件

### 2. 修改 Mapper XML 文件
将 `sys_dormitory_building` 替换为 `my_dormitory_building`

涉及文件：
- `sushe-susheguanli/target/classes/mapper/susheguanli/SysDormitoryBuildingMapper.xml`
- 源文件位置：`sushe-susheguanli/src/main/resources/mapper/susheguanli/SysDormitoryBuildingMapper.xml`

### 3. 修改实体类中的表名注解
将 `@TableName("sys_dormitory_building")` 改为 `@TableName("my_dormitory_building")`

### 4. 其他相关文件
可能需要检查：
- SysDormitoryRoomMapper.xml (`sys_dormitory_room` → `my_dormitory_room`)
- SysStudentMapper.xml (`sys_student` → `my_student`)
- SysAccommodationMapper.xml (`sys_accommodation` → `my_accommodation`)
- SysDormitoryInspectionMapper.xml (`sys_dormitory_inspection` → `my_dormitory_inspection`)
- SysRepairMapper.xml (`sys_repair` → `my_repair`)
